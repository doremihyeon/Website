피케 원단

주로 폴로 셔츠, 카라티 등에도 많이 사용되는 원단으로, 착용감이 좋고 원단 가격이 저렴한 편이라 티셔츠, 원피스 등 각종 의류에 널리 사용됩니다.

장단점

  • 장점 : 신축성이 뛰어나다
  • 단점 : 축률 잡는 것이 까다롭다.

져지

져지는 원단의 앞 뒤가 확연히 구분되며, 디자인에 따라 뒷부분을 앞부분으로 사용하는 경우도 있습니다.

장단점

  • 장점 : 무게가 가볍고 신축성이 뛰어나며, 무게에 비해 두께감이 있다.
  • 단점 : 신축성이 좀 더 떨어지는 특징이 있다.

메리노 울

메리노 울은 개량된 품종의 양모로, 일반 울보다 더 부드럽고 섬세합니다.

장단점

  • 장점 : 부드럽고 섬세하며, 내구성이 뛰어나다
  • 단점 : 신축성과 감촉이 떨어지는 편

일반 울

메일반 양털을 뜻하며, 신축성도 좋고 부드럽습니다. 또한 보온성이 좋고 정전기가 발생하지 않는 장점이 있어 겨울철 의류에 매우 많이 활용되는 원단입니다.

장단점

  • 장점 : 신축성과 감촉이 떨어지는 편
  • 단점 : 보풀과 구김이 잘 생기며, 가격이 비싼 편
